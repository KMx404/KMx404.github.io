Things you need to know

Front-end, Back-end


First, all websites composed of two main parts frontend and backend.

The front-end or the client side cover all the visuals that the user see and interact with..

the backend or the server side is cover the updates and it refers to anything that a user can't see like servers and databases

Things to put in mind:

There's alternatives you should know about !

MEAN (MongoDB, ExpressJS, AngularJS(Angular), NodeJS) is a free open source javascript software stack made for you to make dynamic websites and web applications

What makes MEAN special is that all of its components support programs that are written in JavaScript, MEAN applications can be written in one language for both server-side and client-side execution environments.

in another word, learning javascript with all of these frameworks can save you the trouble of learning multilple other languages


There's also LAMP

(Linux, Apache, MySQL, PHP/Perl/Python) is a very common example of a web service stack, named as an acronym of the names of its original four open-source components: the Linux operating system, the Apache HTTP Server, the MySQL relational database management system (RDBMS), and the PHP programming language. The LAMP components are largely interchangeable and not limited to the original selection. As a solution stack, LAMP is suitable for building dynamic web sites and web applications.
