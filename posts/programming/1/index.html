First of all you should choose the field you like the most ?


Cause the programming world is so big to learn it all aXD ! So you need to choose a specific field such as Web Design , Web Development , Android Apps Development , IOS Apps etc... You need to find the field that inetrets you the most ! And try to focus on ! .

Sources to learn !

You have a lot of places you can learn from such as SoloLearn

SoloLearn has one of my favorite platforms of all time it did really help through my journey

It contains a lot of tutorials of almost all the programming languages you need. It's all just few clicks away


Which programming languages you need to learn ?


In case you choose web design you need to learn :


HTML5,CSS3,Java Script


Those are the most common languages you'll need in your web design career

But you'll need to learn a lot of frameworks/Libraries as well Like Bootstrap, Jquery as libraries


And Angular JS as a JS framework for example ?


Do some projects

Coming to this point now that means you have chosen a specific fiels and you have learned it's programming languages

Now It's time for some projectss . Set a goal to finish a project or two just to make sure you're good enough.


Those projects will help you understand the way these stuff work and what you'll need to learn from now on
